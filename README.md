# Logisimulation

物流をシミュレーションするゲームプラットフォームです。

## ゲーム概要

日本の主要都市間でトラックを使って貨物を配送する物流シミュレーションゲームです。

### 特徴

- 🗾 **10都市マップ**: 東京・大阪・名古屋・札幌・福岡・仙台・広島・金沢・新潟・松山
- 🚛 **トラック管理**: 複数のトラックを所有・管理
- 📦 **注文システム**: ランダムに発生する配送注文
- 💰 **経済システム**: 配送報酬・維持費・トラック購入
- 🗺️ **経路探索**: ダイクストラ法による最短経路自動計算
- ⏱ **ゲーム速度**: 1×〜8×の速度調整

### 操作方法

1. **トラックを選択**: サイドパネルの「トラック」タブでトラックをクリック
2. **注文を割り当て**: 「注文」タブで注文をクリックして配送を開始
3. **トラックを購入**: 資金が ¥500,000 以上の時にトラックを追加購入可能
4. **速度調整**: 画面上部で 1×〜8× の速度でシミュレーションを進める

## 開発・実行

npm run dev
